# Paint Brush

This is a basic paint application which is developed by Java and javax.swing library.
Features for user:

- 7 different colors
- 3 different shapes:
  - Rectangle
  - Elliptical
  - Free Pencil
- Move feature for rectangle and elliptical
- User can draw free with pencil.
